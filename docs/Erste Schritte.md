# Erste Schritte  

Zum Erzeugen eines neuen Textbuchs kann der entsprechende Menuepunkt aufgerufen werden:
##### Textbuch-> Neues Textbuch

Nun kann das neue Dokument benannt werden. Dazu klickt man auf Tit (wie Titel).

![Titel setzen](/img/Titel.png)  
Anschließend navigiert man in den Textbuch-Bereich, setzt den Cursor in die erste Zeile und fügt über das Menü:
##### Bearbeiten->Inhalte einfügen->Unformatierten Text einfügen
den zuvor aus der Fassung der Regieassistenz kopierten Text ein.

:::note[Hinweis]
Es ist wichtig, den Text unformatiert einzufügen, da sonst unter Umständen falsche Seitenformate eingefügt werden, die nicht kompatibel sind. 
:::

Um nun einen Textframe einzufügen, kann der entsprechende Text (Stichwort oder Regieanweisung) markiert werden und dann eine der farbigen Schaltflächen geklickt werden.

![Gelb](/img/Gelb.png)
![Gruen](/img/Gruen.png)
![Blau](/img/Blau.png)
![Rot](/img/Rot.png)
![Cyan](/img/Cyan.png)
![Magenta](/img/Magenta.png)
![Orange](/img/Orange.png)

Dadurch wird der markierte Text in der gewünschten Farbe hervorgehoben und am rechten Rand ein Textframe eingefügt, der bereits den voreingestellten Text enthält.
Dieser Text kann weiter bearbeitet werden.
Um den voreingestellten Text zu ändern, kann das Symbol TXT geklickt werden.

![Standardtext](/img/Txt.png) 

Das öffnet ein Menü zur Bearbeitung der voreingestellten Texte aller Textframes.
