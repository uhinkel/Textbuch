# Seitenformat

Hier kann Einfluss auf das Seitenformat genommen werden.
Es stehen zu Verfügung:
* Vollformat
* Invertiert

Wenn das Vollformat aktiviert wird, wird eine Fußzeile eingefügt, die das aktuelle Kapitel und eine Seitenzahl enthält.

Wenn Invertiert aktiviert wird, wird das Dokument invertiert (also weiß auf schwarz) angezeigt. Dieses Format eignet sich bestens zur Anzeige auf Tablets und schont beim Lesen im Dunkeln die Augen.

Wenn Invertiert deaktiviert wird, wird das Dokument schwarz auf weiß angezeigt und hat links einen Heftrand. Dieses Format eignet sich für den Druck.
