# Cues neu nummerieren

Hier können Cues neu nummeriert werden.

Das Tool funktioniert ähnlich, wie das Tool in SCS. In einer Tabelle werden alle Textframes angezeigt.

Über die Spalte "On" können die zu nummerierenden Textframes ausgewählt werden. Zusätzlich kann die Auswahl auch über das Präfix gesteuert werden. Dieser Präfix wird auch für die Neunummerierung verwendet.
Darüberhinaus kann ein Startwert, eine Schrittweite und das Padding (die Anzahl führender Nullen) gewählt werden.

Mit der Taste "aktualisieren" wird die Vorschau in der Tabelle aktualisiert.

Mit der Taste "anwenden" wird die Vorschau in das aktuelle Dokument übernommen.
