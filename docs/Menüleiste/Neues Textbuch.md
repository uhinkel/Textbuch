# neues Textbuch  

Mit diesem Menüpunkt wird ein neues Textbuch erstellt. Dazu wird eine Vorlagendatei verwendet, die mit der Erweiterung zusammen ausgeliefert wird.
Diese Vorlagendatei enthält alle benötigten Formate und Dokumentenbereiche.


:::note[Hinweis]
Es empfiehlt sich, neue Textbücher immer über diesen Menüpunkt zu erstellen.
Dadurch wird sichergestellt, dass die Erweiterung und die Vorlage kompatibel zueinander sind.
In älteren Dokumenten fehlen möglicherweise wichtige Absatz- oder Seitenformate.
:::
