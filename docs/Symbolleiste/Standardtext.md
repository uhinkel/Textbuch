# Standardtexte
![Standardtexte](/img/Txt.png)  
#### Die Texte, die standardmäßig in den Cueboxen stehen

Hier können Buchstaben oder Wörter hinterlegt werden, die beim Erzeugen einer Cuebox automatisch in dieser Cuebox erscheinen.  
Diese sind wie folgt vorbelegt:



  
* Gelb  Q
* Grün  V
* Rot   Mic
* Blau  HS
* Cyan  L
* Magenta
* Orange
