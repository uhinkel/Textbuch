# Installation  

Zur Installation kann einfach die heruntergeladene Erweiterung (oxt-Datei) geöffnet werden.

Alternativ kann die Erweiterung auch über den Erweiterungsmanager von Libre Office installiert werden:
##### Extras->Erweiterungen
Im Erweiterungsmanager besteht auch die Möglichkeit, die Erweiterungen auf mögliche Updates zu prüfen.
